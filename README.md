# Clue Delayed Gratification

A plugin for players who want to delay the opening of their clue scroll rewards by accumulating a stack of caskets before opening any of them.

## Description

The plugin blocks the **Open** menu option on clue caskets until you have reached a configured minimum stack size. You choose per tier how many caskets you need to collect before you are allowed to open them.

Once you have enough caskets and you start opening them, the plugin tracks your progress. When it detects that you have withdrawn the last of your caskets from the bank and opened them all, it re-locks that tier - so you must build up another stack before opening again.
